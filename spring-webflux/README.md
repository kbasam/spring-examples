## SpringBoot WebFlux demo

## Build and Run
```
gradle clean build test
java -jar build/libs/spring-webflux-1.0.0-SNAPSHOT.jar
```
Go to __index.html__ at ``http://localhost:8080/index.html`` and follow on-screen instructions.
